## ğŸ“© Nesting

To place elements **inside a** `Box`, indent them with a tab. The layout style defined by the parent `Box` applies to all children.

**Example:**

```hcl
<Box flex-h center>
    <Button 0.3>
    <Text 0.7>
```
This creates a horizontally aligned Box with two elements inside: a button and a text block, taking up 30% and 70% width respectively.

If no style is specified, nested content defaults to `flex-v`.
